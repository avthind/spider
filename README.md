#### Key Replacement
The Error Key Replacement program is a **Python** script that processes error messages and updates them based on values extracted from a source file. The script allows users to specify file paths at runtime, making it easy to use with different sets of files without modifying the script itself. The script currently parses source values in a particular format (as shown in the example source.txt and error.txt files) and can be edited to fit other formats as needed.

#### Actionmap 
Actionmap is a web application designed to empower citizens by providing easy access to information about their political representatives, upcoming political events, and relevant news items in their locality. Users can aggregate, share, and view news, fostering an informed and engaged community. The application was built in **Ruby on Rails using MVC and Agile methodologies** as the final project for CS 169A: Software Engineering at UC Berkeley with a group of five students. The project also involved the use of **Cucumber and Capybara** tests.
